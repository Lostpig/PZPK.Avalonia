## PZPK Avalonia版 - Mearii

### Features 功能模块
- core


  - 支持包体类型
    + `.pzpk` PZPK package 标准包体
    + `.pzsv` PZPK single video 单视频文件
    + `.pzpw` PZPK password book 密码本文件
    + `.pztx` PZPK text 可编辑文本文件

- desktop
- 登录界面 `绑定sqlite数据库`
- 主界面
  - 浏览
    - 索引浏览
    - 包内文件浏览
  - 创建
  - 密码本
  - 设置
- 主要窗口
  - 创建包体窗口
  - 图片浏览窗口
  - 视频浏览窗口
  - 文本编辑窗口
- 次要窗口
  - 打开文件输入密码窗口
  - 创建包体设置窗口
  - 索引扫描进度窗口
  - 打包/解包 进度窗口
  - 密码本管理窗口
