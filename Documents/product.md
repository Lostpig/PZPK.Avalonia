## PZPK Avalonia版

### Features 功能模块
- core
  - 支持包体类型
    + `.pzpk` PZPK package 标准包体
    + `.pznt` PZPK notebook 笔记本

- desktop
- 主界面
  - 浏览 `打开及浏览PZPK包内文件`
  - 创建 `创建PZPK包`
  - 笔记 `创建/查看/编辑笔记本文件`
  - 仓库 `管理包文件索引(基于SQLite)`
  - 设置
- 窗口
  - 图片浏览窗口
  - 视频浏览窗口
