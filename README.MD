## PZPK Avalonia版
PZPK 包管理桌面应用程序Avalonia版

基于*Avalonia*开发，使用了修改自*Avalonia.Markup.Declarative*的*PZ.RxAvalonia*

使用了*SukiUI*组件库 

使用*Rx.Net*管理状态

## 功能模块
  + `pzpk` 包浏览 - 图片/视频/文本预览，文件提取
  + `pzpk` 打包
  + `pznt` 笔记功能
  + 仓库管理
  + 设置 - 多语言/主题
